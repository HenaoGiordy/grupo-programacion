# 👨‍💻 Bienvenidos al Grupo de Programación

> *"El código es poesía que las máquinas pueden leer."*

---

## ¿Qué es este espacio?

Este es el blog oficial del grupo de programación. Aquí encontrarás todo lo que necesitas para seguir el curso: las **clases publicadas**, las **tareas asignadas**, recursos de apoyo y más.

Cada entrada nueva será una clase o una tarea. Revisa este espacio con regularidad para no perderte nada.

---

## ¿De qué trata el curso?

Este grupo está pensado para aprender a programar desde cero (o reforzar lo que ya sabes), con un enfoque práctico y progresivo. Las sesiones combinan teoría clara con ejercicios reales para que puedas aplicar lo aprendido de inmediato.

A lo largo del curso vamos a explorar:

- Fundamentos de la programación y lógica computacional
- Estructuras de datos y algoritmos básicos
- Proyectos pequeños y retos de código
- Buenas prácticas y hábitos de un buen programador

---

## ¿Cómo funciona?

### 📚 Clases
Cada clase se publicará aquí como una entrada del blog. Incluirá:
- El tema tratado
- Explicaciones y ejemplos de código
- Recursos y lecturas recomendadas

### 📝 Tareas
Las tareas se publicarán junto a la clase correspondiente o como entradas independientes. Cada tarea tendrá:
- Una descripción clara del ejercicio
- La fecha límite de entrega
- Los criterios de evaluación

---

## Reglas del grupo

1. **Respeto ante todo** — Este es un espacio seguro para aprender. No hay preguntas tontas.
2. **Participación activa** — Intenta resolver las tareas antes de pedir ayuda.
3. **Comparte tu proceso** — A veces aprendemos más del error que del acierto.
4. **Entrega a tiempo** — Las fechas límite existen por una razón.
5. **Código propio** — Está bien consultar recursos, pero el código que entregues debe ser tuyo.
